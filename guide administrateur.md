# 📋 Guide maintenance — Boîtier de Satisfaction

## 1. Présentation

Le boîtier de satisfaction est un appareil simple et intuitif permettant d'exprimer votre niveau de satisfaction concernant l'ambiance de la cafétéria en appuyant sur l'un des deux boutons colorés.

---

## 2. Description du boîtier

| Élément | Description |
|---|---|
| 🟢 Bouton vert | Permet la navigation dans les options - Exprime une satisfaction **positive** |
| 🔴 Bouton rouge | Permet la validation - Exprime une satisfaction **négative** |
| 💡 Écran | Retour utilisateur |

---

## 3. Utilisation

### Étape 1 — Vérifier que le boîtier est actif
L'écran ou la LED doit être **allumé(e)**. Si ce n'est pas le cas, le boîtier n'est pas prêt à recevoir votre vote.

### Étape 2 — Appuyer sur les boutons pour séletionner le mode d'utilisation et exprimer votre ressenti
- Appuyer sur un bouton pour vous utiliser le boitier
- Appuyer sur le **bouton vert** pour changer de mode d'utilisation
- Appuyer sur le **bouton rouge** pour séléctionner le mode d'utilisation
- Appuyez sur le **bouton vert** pour modifier le nom de l'appareil (celui-ci reste en mémoire si l'appareil est débranché)
- Appuyez sur le **bouton rouge** pour sauvegarder la sélection
